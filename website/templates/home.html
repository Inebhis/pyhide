3rd year Computer Science project - Steganography


This showcase site is used as a portal for the demonstration. The proposed project is a steganography tool, principle explained on the dedicated page.
It is divided into two blocks: : hide and unhide.
Hide is a form where we can enter the text to be hidden and the image used as a receptacle for the message.
Unhide is a form where only an image can be loaded to discover (if it exists), a message to display.
